Enter two numbers









Problem Statement: You have been tasked with creating an HTML page that allows the user to perform basic math operations using JavaScript. Your page should include input fields for two numbers and display the results of the following operations:

1. Addition: Add the two numbers and display the result.

2. Subtraction: Subtract the second number from the first number and display the result.

3. Multiplication: Multiply the two numbers and display the result

4. Division: Divide the first number by the second number and display the result

Your page should handle positive and negative numbers. Additionally, it should display an error message if the user enters an invalid input (e.g., non-numeric input).
